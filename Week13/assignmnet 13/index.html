About the Neon Genesis Evangelion Web App

×

This web app was created as part of a Bootstrap tutorial and themed around the popular anime series Neon Genesis Evangelion. It features Bootstrap components such as containers, buttons, responsiveness, modals, tooltips, carousels, and accordions.

Thank you for checking out this web app!

Trivia

The name of the organization is NERV.
The name of the main protagonist is Shinji Ikari.
The name of the pilot of Evangelion Unit-02 is Asuka Langley Soryu.
